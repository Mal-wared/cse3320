Detailed Description

Get experience creating and running concurrent processes and threads.

Part 1: You will create processes, run processes in parallel, and pass information between processes.
The data to be processed are “tuples” which are numbers and associated text strings, that is lines of CSV separated values, we wish to sort these.

Part 2: You will create a process and threads, run in parallel, and pass information between threads.
You will use synchronization operations to avoid overwriting shared resources.
